# tf.keras.models.save_model

## 简介

```python
tf.keras.models.save_model(
    model,
    filepath,
    overwrite=True,
    include_optimizer=True,
    save_format=None,
    signatures=None,
    options=None,
    save_traces=True
)
```

将模型保存为 TF SavedModel 或 HDF5 文件。
